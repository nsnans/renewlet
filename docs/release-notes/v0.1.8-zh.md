# Renewlet v0.1.8

Renewlet v0.1.8 带来家庭共享费用分摊（[zhiyingzzhou/renewlet#32](https://github.com/zhiyingzzhou/renewlet/pull/32)，感谢 [@Shy0o](https://github.com/Shy0o)）。给一项订阅加上共享成员后，Renewlet 会按你的分摊方式显示自己的实际份额、家庭成员贡献和可回收金额；同时，Cloudflare 一键部署的后续升级入口、货币选择和长筛选列表也更顺手。

[English ->](./v0.1.8-en.md)

---

## 升级前请先读

Docker 用户从 v0.1.7 升级不需要手动迁移。

Cloudflare 用户请按现有部署流程发布；本版本包含家庭共享费用分摊所需的数据迁移。

Cloudflare 一键部署用户以后从生成仓库运行 `Sync Renewlet Upstream` 升级，不要重新点一键部署按钮。旧生成仓库如果还没有这个入口，按 [Cloudflare 部署文档](https://github.com/zhiyingzzhou/renewlet/blob/main/docs/cloudflare-workers-deploy.zh-CN.md#旧一键部署用户) 一次性补上。

---

## 新功能

- 家庭共享费用分摊：订阅可添加共享成员，并选择平均分摊或自定义金额。
- 成员可使用不同货币；订阅卡片和详情会显示你的份额、家庭成员贡献和可回收金额。
- 分摊信息会随订阅正常保存，导入和导出时一起保留。

---

## 优化改进

- Cloudflare 一键部署新增后续升级入口，生成仓库可手动同步 Renewlet 上游版本。
- 货币名称和代码更清楚，常用货币更容易找到。
- 货币选择器会跟随设置里的排序；旧订阅正在使用但已禁用的货币只回显，不再作为新选择项。
- 分类和标签较多时，订阅筛选弹窗会在列表区域滚动，底部操作不再被挤出视野。
